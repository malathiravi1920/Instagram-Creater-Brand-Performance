#  Installation

Clone the repository

```bash
git clone https://github.com/yourusername/my_dashboard_app.git
```

Go to the project folder

```bash
cd my_dashboard_app
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
